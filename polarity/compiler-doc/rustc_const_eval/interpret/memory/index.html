The memory subsystem.

Generally, we use Pointer to denote memory addresses. However, some operations have a “size”-like parameter, and they take Scalar for the address because if the size is 0, then the pointer can also be a (properly aligned, non-null) integer. It is crucial that these operations call check_align before short-circuiting the empty case!
