pub fn into_raw_command(self) -> Command

In very rare circumstances, you may need a e.g. bare_rustc() or bare_rustdoc() with host runtime libs configured, but want the underlying raw std::process::Command (e.g. for manipulating pipes or whatever). This function will consume the command wrapper and extract the underlying std::process::Command.

Caution: this will mean that you can no longer use the convenience methods on the command wrapper. Use as a last resort.
