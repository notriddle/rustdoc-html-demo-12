pub struct CompilerCtxt<'tcx, B: Bridge> {
    pub tcx: TyCtxt<'tcx>,
    _marker: PhantomData<B>,
}
Expand description

Provides direct access to rustc’s internal queries.

CompilerInterface must go through this context to obtain internal information.
