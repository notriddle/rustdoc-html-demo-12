Crate that implements what will become the rustc side of rustc_public.

This crate serves as a proxy for making calls to rustc queries.

This crate is not intended to be invoked directly by users. This crate is the public API of rustc that will be invoked by the rustc_public crate.

For more information see https://github.com/rust-lang/project-stable-mir
