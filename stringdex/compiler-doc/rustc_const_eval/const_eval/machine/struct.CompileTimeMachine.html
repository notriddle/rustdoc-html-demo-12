Expand description

Extra machine state for CTFE, and the Machine instance.

Fieldsยง

ยงnum_evaluated_steps: usize

The number of terminators that have been evaluated.

This is used to produce lints informing the user that the compiler is not stuck. Set to usize::MAX to never report anything.

ยงstack: Vec<Frame<'tcx>>

The virtual call stack.

ยงcan_access_mut_global: CanAccessMutGlobal

Pattern matching on consts with references would be unsound if those references could point to anything mutable. Therefore, when evaluating consts and when constructing valtrees, we ensure that only immutable global memory can be accessed.

ยงcheck_alignment: CheckAlignment

Whether to check alignment during evaluation.

ยงstatic_root_ids: Option<(AllocId, LocalDefId)>

If Some, we are evaluating the initializer of the static with the given LocalDefId, storing the result in the given AllocId. Used to prevent accesses to a staticโs base allocation, as that may allow for self-initialization loops.
