Expand description

Deeply normalize all aliases in value. This does not handle inference and expects its input to be already fully resolved.

Additionally takes a list of universes which represents the binders which have been entered before passing value to the function. This is currently needed for normalize_erasing_regions, which skips binders as it walks through a type.

This returns a set of stalled obligations involving coroutines if the typing mode of the underlying infcx has any stalled coroutine def ids.
