Enum Attribute

Source 
pub enum Attribute {
    Parsed(AttributeKind),
    Unparsed(Box<AttrItem>),
}

Variants§

§

Parsed(AttributeKind)

A parsed built-in attribute.

Each attribute has a span connected to it. However, you must be somewhat careful using it. That’s because sometimes we merge multiple attributes together, like when an item has multiple repr attributes. In this case the span might not be very useful.

§

Unparsed(Box<AttrItem>)

An attribute that could not be parsed, out of a token-like representation. This is the case for custom tool attributes.
